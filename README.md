# Общие задания

---

## Задание 1

Напишите программу, которая запрашивает у пользователя целое число и определяет, является ли оно чётным или нечётным. Выведите соответствующее сообщение на экран.

**Требования:**

- Использовать оператор ветвления `if-else`.
- Для проверки чётности использовать оператор остатка от деления `%`.

## Задание 2

Напишите программу, которая запрашивает у пользователя два числа и выводит наименьшее из них. Если числа равны, выведите сообщение, что числа равны.

**Требования:**

- Использовать тернарное выражение для определения наименьшего числа.
- Обеспечить корректную обработку ситуации, когда числа равны.

## Задание 3

Напишите программу, которая принимает от пользователя дату (день и месяц) и определяет, к какому времени года она относится. Выведите название сезона.

**Требования:**

- Использовать операторы `if-elif-else` для определения сезона.
- Считайте, что:
    - **Весна:** с 1 марта по 31 мая
    - **Лето:** с 1 июня по 31 августа
    - **Осень:** с 1 сентября по 30 ноября
    - **Зима:** с 1 декабря по 28 февраля

## Вариант 2

### Задание 4

Напишите программу, которая запрашивает у пользователя три числа и определяет, существует ли треугольник с такими сторонами. Выведите сообщение о результате.

**Требования:**

- Использовать неравенство треугольника для проверки возможности существования треугольника.
- Использовать логические операции для объединения условий.

### Задание 5

Напишите программу, которая принимает от пользователя целое число и выводит его абсолютное значение без использования встроенной функции `abs()`. Используйте тернарное выражение.

**Требования:**

- Использовать тернарное выражение для расчёта абсолютного значения.
- Не использовать функцию `abs()`.
